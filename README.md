# Notes App

This is a simple note taking Android App including:
 MVVM,
 RecyclerView,
 Navigation,
 ViewModel,
 LiveData,
 ViewBinding,
 Room,
 Coroutines,
 Flow
 
 
 ## Features

- Add notes
- Edit notes
- Delete notes
